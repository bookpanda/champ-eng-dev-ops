# ChAMP SWE Track DevOps Assignment

## Installation
- git clone https://github.com/bookpanda/champ-eng-go-crud.git
- docker compose up
- Test with Postman

### Product service

- `GET /product/:id`
- `POST /product`
- `GET /healthcheck`

### User service

- `GET /user/:id`
- `GET /user/:id/product`
- `POST /user/:id/product`
- `POST /user`
- `GET /healthcheck`
